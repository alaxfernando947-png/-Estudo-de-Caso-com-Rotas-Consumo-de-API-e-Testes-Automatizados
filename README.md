#📘 Meu Estudo de Caso — Aplicação Web + Mobile

Este repositório contém um estudo de caso completo desenvolvido com foco em boas práticas, componentização, interface responsiva, e utilização de tecnologias modernas tanto no front-end web (Vite + React) quanto no mobile (Expo + React Native).

O objetivo é demonstrar na prática como estruturar um projeto real, organizado e escalável, usando ferramentas atuais do ecossistema JavaScript.
